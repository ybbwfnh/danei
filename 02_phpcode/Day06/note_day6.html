关联数组使用另外一种方式,foreach来循环 7.数组函数库 count/sizeof 获得数组或对象的个数 join/implode: 将一纬数组转换成字符串 explode :将字符串 转为一维数组 $arr = 'a*b*c*d'; $newarr = explode('*',$arr),2; 第三个是参数.一共要几个值 8.指针函数 用法echo each($arr); current :返回数组指针指向当前的成员值,并且返回該值. reset:返回数组指针指向第一个成员值,并且返回該值. next:返回数组指针指向下一个成员值,并且返回該值. prev:返回数组指针指向上一个成员值,并且返回該值. end:返回数组指针指向最后一个成员值,并且返回該值. each :返回 以键和值组成的数组,然后指针向下一位 each 最好跟current 套用,才合适.显示并且指针下移 echo current(each($arr));否则很乱. list: 把数组中的值赋给一些变量. 9. in_array: 查找值在数组中是否存在,返回Bool值. $var = 'b'; $arr = array('a','b','c'); 例如: var_dump(in_array($var,$arr)); 具体看手册. 包括松散比较和严格比较. var_dump(in_array($var,$arr)true); array_key_exists: 检测键名称是否存在于数组中. 10. array_push: 向数组中尾部添加一个或多个成员,并返回数组长度. array_unshift:向数组中头部添加一个或多个成员,并返回数组长度. array_php: 向数组中尾部删除一个或多个成员,并返回数组长度. array_shift:向数组中头部删除一个或多个成员,并返回数组长度 array_map: 将回调函数作用到给定函数 二维数组(多维数组) 方法一: 嵌套关系,array('a',array('b','c')); 重要: 定义二维数组(索引+关联) 外索引,内关联 $arr = array( 0=> array('name'=>'张三','age'=>12,'add'=>'北京'), 1=> array('name'=>'李四','age'=>13,'add'=>'天津'), 2=> array('name'=>'王五','age'=>15,'add'=>'上海'), ) print_r($arr) 方法二: 添加发 $arr2 = array();//初始化 $arr2[] = array('name'=>'张三','age'=>12,'add'=>'北京'); $arr2[] = array('name'=>'李四','age'=>13,'add'=>'天津'); $arr2[] = array('name'=>'王五','age'=>15,'add'=>'上海');
